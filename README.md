# archeryosa-events-calendar
Apps Script to scrape events from ArcheryOSA into a Google Calendar.

ArcheryOSA is publicly accessible at [archeryosa.com](https://archeryosa.com).

Contribute to ArcheryOSA at [GitHub](https://github.com/steveclifton/ArcheryOSA-v.3).
